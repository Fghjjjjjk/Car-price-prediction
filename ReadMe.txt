# Car Price Prediction

This project uses a Random Forest model to predict car prices based on features like mileage, production year, engine volume, and model.

## Files

- `car_price_prediction.Rmd`: R Markdown file containing the code and analysis.
- `cardata.csv`: Dataset used for training and testing the model.
- `README.md`: Project documentation.
